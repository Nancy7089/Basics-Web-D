CSS BASICS

There are three ways to use css

  1. Inline : Inside an HTML tag
  2. Internal : Inside a style tag in HTML
  3. External : In a .css file

Common CSS properties:

  • color : text color
  • background-color : background color background-color: rgba(0, 0, 255, 0.5);
  • font-size : text size
  • font-family : font type , add backup fonts ,"Roboto", Helvetica, sans-serif;
  • font-weight: boldness, bold,700
  • text-align : align text
  • border : border around an element : 1px solid black;
  • margin : space outside the element
  • padding : space inside the element

CSS Selectors

  1. Element selector, targets all elements of a specific type
  2. Class selector(.), targets elements with specific class attribute
  3. ID selector(#), targets a unique element with an id, IDs should be unique on the page.
  4. Universal selector(*), targets all elemnts on page
  5. Group selector(h1,h2,p{}), style multiple elements at once
  6. Nested selector(div,p{}), all p tags inside div

CSS Positioning

  1. static:normal flow
  2. relative:moves relative to its normal position
  3. absolute:Moves relative to its closest positioned ancestor
  4. fixed:Stays fixed to the screen, like a sticky navbar
  5. sticky:Switches between relative and fixed based on scroll position
